# 🦊🐰 Foxes and Rabbits - Simulation

Интерактивная симуляция экосистемы с лисами, кроликами и травой на сетке 50×50. Проект демонстрирует взаимодействие между хищниками и добычей в реальном времени.

## 📋 Описание

Это веб-приложение моделирует простую экосистему, где:
- **🐰 Кролики** — ищут траву для еды и убегают от лис
- **🦊 Лисы** — охотятся на кроликов
- **🌾 Трава** — служит пищей для кроликов

Каждый объект движется по 50×50 сетке и взаимодействует с окружающей средой согласно простым правилам выживания.

## 🎮 Как использовать

1. Откройте `index.html` в браузере
2. Нажмите кнопку **"Запустить"** для начала симуляции
3. Наблюдайте за движением лис, кроликов и травы
4. Нажмите **"Остановить"** для паузы симуляции

## 🎯 Основные компоненты

### Классы сущностей

- **Entity** — базовый класс для всех объектов с методами движения и поиска
- **Fox** — хищник, охотится на кроликов
- **Rabbit** — травоядное, ищет еду и избегает опасност��
- **Grass** — растительность, пища для кроликов

### Алгоритмы поведения

**Лиса:**
- Ищет ближайшего кролика в радиусе видимости
- При расстоянии = 1 клетка — съедает кролика
- Иначе движется в направлении добычи

**Кролик:**
- Если лиса ближе 4 клеток — убегает в противоположном направлении
- Если трава рядом (1 клетка) — съедает её
- Иначе движется к ближайшей траве

## ⚙️ Ключевые функции

| Функция | Описание |
|---------|---------|
| `goTo(direction)` | Движение сущности в указанном направлении с проверкой границ |
| `lookUp(target, place)` | Поиск целевого объекта в радиусе видимости |
| `render(arr)` | Отрисовка всех сущностей на сетке |
| `startEmul()` | Основной цикл симуляции с интервалом 500ms |

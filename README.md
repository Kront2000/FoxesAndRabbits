# 🦊🐰 Foxes and Rabbits - Simulation

Интерактивная симуляция экосистемы с лисами, кроликами и травой на сетке 50×50. Проект демонстрирует взаимодействие агентов в простой экосистеме.

> **📌 Заметка:** Это учебный проект, написанный за пол дня для закрепления знаний по базовому нативному JavaScript. Код содержит множество недочётов и мест для улучшения: грязные функции, слабая инкапсуляция, отсутствие разделения ответственности. Проект создан в образовательных целях и демонстрирует начальный уровень понимания основ JS.

## 📋 Описание

Это веб-приложение моделирует простую экосистему, где:
- **🐰 Кролики** — ищут траву для еды и убегают от лис
- **🦊 Лисы** — охотятся на кроликов
- **🌾 Трава** — служит пищей для кроликов и появляется автоматически каждые 4 такта

Каждый объект движется по 50×50 сетке и взаимодействует с окружающей средой согласно простым правилам выживания.

## 🎮 Как использовать

1. Откройте `index.html` в браузере
2. Настройте начальные параметры через форму:
   - **Количество лис** — контролирует популяцию хищников
   - **Количество кроликов** — контролирует популяцию травоядных
   - **Количество травы** — контролирует начальное количество пищи
   - **Время одного тика** — скорость симуляции (в миллисекундах)
3. Нажмите кнопку **"Запустить"** для начала симуляции
4. Наблюдайте за движением лис, кроликов и травы в реальном времени
5. **Статистика популяций** обновляется динамически
6. Нажмите **"Остановить"** для паузы, или **"Перезапустить"** для сброса симуляции

## 🎯 Основные компоненты

### Классы сущностей

- **Entity** — базовый класс для всех объектов с методами движения, поиска и взаимодействия
- **Fox** — хищник, охотится на кроликов (наследует Entity)
- **Rabbit** — травоядное, ищет еду и избегает опасности (наследует Entity)
- **Grass** — растительность, пища для кроликов (наследует Entity)

### Алгоритмы поведения

**Лиса:**
- Ищет ближайшего кролика в радиусе видимости с помощью `lookUp()`
- При расстоянии = 1 клетка — съедает кролика методом `eat()`
- При съедении 2 кроликов — воспроизводит потомство (`giveBirth()`)
- Иначе движется в направлении добычи методом `goTo()`

**Кролик:**
- Если лиса ближе 4 клеток — убегает в противоположном направлении
- Если трава рядом (1 клетка) — съедает её
- При съедении 2 травинок — воспроизводит потомство
- Иначе движется к ближайшей траве

**Трава:**
- Автоматически возникает каждые 4 такта симуляции
- Может быть съедена кроликами

## ⚙️ Ключевые функ��ии

| Функция | Описание |
|---------|---------|
| `goTo(direction)` | Движение сущности в указанном направлении с проверкой границ |
| `lookUp(target, place)` | Поиск целевого объекта в радиусе видимости, возвращает расстояние и направление |
| `eat(target)` | Поедание целевого существа и увеличение счётчика съеденного |
| `giveBirth()` | Воспроизводство потомства данного вида |
| `isEmptyHere(x, y)` | Проверка доступности клетки сетки |
| `render(arr)` | Отрисовка всех сущностей на сетке и обновление статистики |
| `startEmul()` | Основной цикл симуляции с интервалом в `tickTime` миллисекунд |
| `resetStatistics(arr)` | Расчёт и отображение процентного состава популяций |
| `populate(fox, rabbit, grass)` | Инициализация начального населения |

## 📊 Параметры Симуляции

- **Размер сетки:** 50×50 клеток
- **Интервал обновления:** настраивается пользователем (по умолчанию 500ms)
- **Начальное население:** настраивается через форму (по умолчанию 2 лисы, 4 кролика, 10 травинок)
- **Возрождение травы:** каждые 4 такта появляется 1 новая травинка
- **Репродукция:** животные размножаются после съедения 2 целевых объектов
